# Philippine seed strategy

Status: fixture-only; no live or official nutrition data is included.

This directory is reserved for versioned Philippine alias and structural fixtures that can later be transformed into seed rows after the database schema is stable. The JSON file here is test/example material, not a migration and not a production nutrition source.

The repository does **not** claim access to, licensing for, or redistribution rights for PhilFCT or DOST-FNRI composition data. Before any such dataset is added, the team must verify current access, licensing, attribution, and redistribution requirements. Until then, the provider contract remains present but blocked.

Nutrition values for restaurant products are deliberately absent. Menu aliases and mock combo component structures do not establish official nutrition availability. Production imports must preserve source, Philippine market evidence, source version, and the exact observation used at confirmation.
